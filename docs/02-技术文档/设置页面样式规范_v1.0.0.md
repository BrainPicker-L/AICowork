## 一、设计原则

### 1.1 核心原则

| 原则 | 描述 |
|------|------|
| **一致性** | 所有设置区域使用相同的视觉语言和交互模式 |
| **清晰性** | 信息层级明确，用户能快速找到所需功能 |
| **高效性** | 减少操作步骤，提供快捷路径 |
| **容错性** | 提供清晰的错误提示和恢复机制 |

### 1.2 设计目标

- 降低学习成本，用户熟悉一个区域后能快速上手其他区域
- 减少视觉噪音，使用留白和分组提高可读性
- 提供即时反馈，让用户了解操作结果

## 二、色彩系统

### 2.1 色彩概览

```css
/* 主题色 - 强调色 */
--accent: #0066cc;
--accent-hover: #0052a3;
--accent-light: #e6f0ff;

/* 中性色 */
--ink-900: #1a1a1a;  /* 主要文字 */
--ink-700: #4a4a4a;  /* 次要文字 */
--ink-100: #999999;  /* 弱化文字 */

/* 背景色 */
--surface: #ffffff;
--surface-secondary: #f5f5f5;
--surface-tertiary: #ebebeb;

/* 功能色 */
--success: #10b981;
--success-light: #d1fae5;
--error: #ef4444;
--error-light: #fee2e2;
--warning: #f59e0b;
--warning-light: #fef3c7;
```

### 2.2 文字色彩

| 用途 | 色彩 | Tailwind 类 | 使用场景 |
|------|------|-------------|----------|
| 主要文字 | `#1a1a1a` | `text-ink-900` | 页面标题、卡片标题、重要内容 |
| 次要文字 | `#4a4a4a` | `text-ink-700` | 正文内容、按钮文字 |
| 弱化文字 | `#999999` | `text-muted` | 描述文字、提示信息、占位符 |
| 极弱文字 | `#cccccc` | `text-muted-light` | 辅助说明、禁用状态文字 |
| 强调文字 | `#0066cc` | `text-accent` | 链接、高亮文字、重要图标 |
| 成功文字 | `#10b981` | `text-success` | 成功提示 |
| 错误文字 | `#ef4444` | `text-error` | 错误提示 |

### 2.3 背景色彩

| 用途 | 色彩 | Tailwind 类 | 使用场景 |
|------|------|-------------|----------|
| 主背景 | `#ffffff` | `bg-surface` | 卡片、输入框背景 |
| 次背景 | `#f5f5f5` | `bg-surface-secondary` | 次级卡片、代码块、区域分隔 |
| 三级背景 | `#ebebeb` | `bg-surface-tertiary` | 悬停状态、按钮悬停 |
| 强调背景（浅） | `rgba(0, 102, 204, 0.05)` | `bg-accent/5` | 选中卡片背景 |
| 强调背景（中） | `rgba(0, 102, 204, 0.1)` | `bg-accent/10` | 标签背景 |
| 成功背景 | `#d1fae5` | `bg-success-light` | 成功提示背景 |
| 错误背景 | `#fee2e2` | `bg-error-light` | 错误提示背景 |

## 三、字体系统

### 3.1 字体族

```css
/* 系统默认字体栈 */
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto,
             "Helvetica Neue", Arial, sans-serif;
```

### 3.2 字号与字重

| 级别 | 字号 | 字重 | 行高 | Tailwind 类 | 使用场景 |
|------|------|------|------|-------------|----------|
| **H1** | 24px | 600 (Semibold) | 1.2 | `text-2xl font-semibold` | 页面主标题 |
| **H2** | 18px | 500 (Medium) | 1.3 | `text-lg font-medium` | 区域标题 |
| **H3** | 14px | 500 (Medium) | 1.4 | `text-sm font-medium` | 卡片标题、表单标签 |
| **Body** | 14px | 400 (Regular) | 1.5 | `text-sm` | 正文内容 |
| **Small** | 12px | 500 (Medium) | 1.5 | `text-xs font-medium` | 辅助说明、标签 |
| **Caption** | 12px | 400 (Regular) | 1.5 | `text-xs` | 提示文字、占位符 |
| **Code** | 12px | 400 (Regular) | 1.4 | `text-xs font-mono` | 代码、命令路径 |

### 3.3 字体使用规范

```typescript
// ✅ 正确的字体使用
<h1 className="text-2xl font-semibold text-ink-900">
  页面标题
</h1>

<p className="text-sm text-muted">
  描述文字
</p>

<label className="text-xs font-medium text-muted">
  表单标签
</label>

// ❌ 错误的字体使用
<h1 className="text-lg text-ink-900">  // 字号太小
  页面标题
</h1>

<p className="text-base text-muted">   // 字号太大
  描述文字
</p>
```

### 3.4 文字处理规则

| 场景 | 处理方式 | Tailwind 类 |
|------|----------|-------------|
| 单行省略 | 超出显示省略号 | `truncate` |
| 多行省略 | 限制行数并显示省略号 | `line-clamp-2` |
| 换行控制 | 允许换行 | 默认 |
| 不换行 | 强制单行 | `whitespace-nowrap` |
| 断词 | 长单词断行 | `break-all` / `break-words` |

## 四、间距系统

### 4.1 间距标尺

```css
/* 基础间距单位: 4px */
--spacing-1: 4px;    /* 极小间距 */
--spacing-2: 8px;    /* 小间距 */
--spacing-3: 12px;   /* 中小间距 */
--spacing-4: 16px;   /* 中间距 */
--spacing-5: 20px;   /* 中大间距 */
--spacing-6: 24px;   /* 大间距 */
--spacing-8: 32px;   /* 极大间距 */
```

### 4.2 间距使用场景

| 场景 | 间距值 | Tailwind 类 | 说明 |
|------|--------|-------------|------|
| Section 内元素 | 24px | `space-y-6` | 主要区域垂直间距 |
| 内容区域分组 | 16px | `space-y-4` | 内容卡片之间的间距 |
| 表单字段组 | 16px | `space-y-4` | 表单区域之间的间距 |
| 表单字段内 | 6px | `gap-1.5` | 标签与输入框间距 |
| 卡片内边距 | 16px | `p-4` | 卡片内容与边框间距 |
| 按钮组间距 | 12px | `gap-3` | 按钮之间的间距 |
| 标题下边距 | 8px | `mt-2` | 标题与描述文字间距 |
| 图标文字间距 | 8px | `gap-2` | 图标与文字间距 |
| 极小间距 | 4px | `gap-1` | 紧密元素间距 |

### 4.3 间距可视化

```
┌──────────────────────────────────────────────────────┐
│ Section (space-y-6: 24px)                            │
│                                                       │
│   ┌─────────────────────────────────────────────┐    │
│   │ Header                                       │    │
│   │   H1: text-2xl font-semibold                 │    │
│   │   P: mt-2 text-sm                           │    │
│   └─────────────────────────────────────────────┘    │
│                                                       │
│   ↓ 24px                                              │
│                                                       │
│   ┌─────────────────────────────────────────────┐    │
│   │ Content Space (space-y-4: 16px)             │    │
│   │                                             │    │
│   │   Card 1 (p-4)                              │    │
│   │     ↓ 16px                                  │    │
│   │   Card 2 (p-4)                              │    │
│   │     ↓ 16px                                  │    │
│   │   Card 3 (p-4)                              │    │
│   │                                             │    │
│   └─────────────────────────────────────────────┘    │
│                                                       │
└──────────────────────────────────────────────────────┘
```

### 4.4 Padding vs Gap 规则

```typescript
// ✅ 使用 gap: flex/grid 容器内元素间距
<div className="flex gap-3">
  <button>按钮1</button>
  <button>按钮2</button>
</div>

// ✅ 使用 space-y: 垂直堆叠元素间距
<div className="space-y-4">
  <div>卡片1</div>
  <div>卡片2</div>
</div>

// ✅ 使用 p-4/p-6: 容器内边距
<div className="p-4">
  内容...
</div>
```

## 五、组件规范

### 5.1 容器组件

#### Section 容器

```typescript
// 最外层容器
<section className="space-y-6">
  <header>...</header>
  <div>...</div>
  <aside>...</aside>
</section>
```

**样式类**: `space-y-6`
**用途**: 包裹整个设置区域

#### Header 容器

```typescript
// 带操作按钮的 Header
<header className="flex items-center justify-between">
  <div>
    <h1 className="text-2xl font-semibold text-ink-900">
      {t('section.title')}
    </h1>
    <p className="mt-2 text-sm text-muted">
      {t('section.description')}
    </p>
  </div>
  {viewMode === 'list' && (
    <button className="rounded-xl border border-ink-900/10 bg-surface px-4 py-2 text-sm text-ink-700 hover:bg-surface-tertiary transition-colors">
      添加
    </button>
  )}
</header>
```

**样式类**: `flex items-center justify-between`
**布局**: 左侧标题+描述，右侧操作按钮

#### 卡片容器

```typescript
// 标准卡片
<div className="p-4 rounded-xl border border-ink-900/10 bg-surface">
  <h3 className="text-sm font-medium text-ink-900">标题</h3>
  <p className="mt-1 text-xs text-muted">描述</p>
</div>

// 次级卡片（提示框）
<div className="p-4 rounded-xl bg-surface-secondary border border-ink-900/5">
  <p className="text-xs text-muted">
    <strong>提示：</strong>提示内容
  </p>
</div>
```

| 类型 | 样式类 | 用途 |
|------|--------|------|
| 标准卡片 | `p-4 rounded-xl border border-ink-900/10 bg-surface` | 内容卡片 |
| 次级卡片 | `p-4 rounded-xl bg-surface-secondary border border-ink-900/5` | 提示框 |

## 十一、代码示例

### 11.1 完整设置区域模板

```typescript
/**
 * @author      Alan
 * @copyright   AGCPA v3.0
 * @created     2026-01-21
 * @updated     2026-01-21
 * @Email       None
 *
 * 功能区域模板
 */

import { useState, useEffect } from "react";
import { useTranslation } from "react-i18next";

type ViewMode = 'list' | 'create' | 'edit';

interface Item {
  id: string;
  name: string;
  description: string;
}

export function FeatureSection() {
  const { t } = useTranslation();
  const [viewMode, setViewMode] = useState<ViewMode>('list');
  const [items, setItems] = useState<Item[]>([]);
  const [loading, setLoading] = useState(false);

  // 表单状态
  const [itemName, setItemName] = useState('');
  const [itemDescription, setItemDescription] = useState('');
  const [saving, setSaving] = useState(false);
  const [error, setError] = useState<string | null>(null);
  const [success, setSuccess] = useState(false);

  // 加载数据
  const loadItems = async () => {
    setLoading(true);
    try {
      // 加载逻辑
      const data = await fetchItems();
      setItems(data);
    } catch (err) {
      console.error('Failed to load items:', err);
    } finally {
      setLoading(false);
    }
  };

  useEffect(() => {
    loadItems();
  }, []);

  // 创建/编辑
  const handleSave = async () => {
    // 验证
    if (!itemName.trim()) {
      setError('名称不能为空');
      return;
    }

    setError(null);
    setSaving(true);

    try {
      const result = await saveItem({
        name: itemName.trim(),
        description: itemDescription.trim(),
      });

      if (result.success) {
        setSuccess(true);
        setTimeout(() => setSuccess(false), 2000);
        // 重置表单
        setItemName('');
        setItemDescription('');
        // 重新加载列表
        await loadItems();
      } else {
        setError(result.error || '保存失败');
      }
    } catch (err) {
      setError('保存失败');
    } finally {
      setSaving(false);
    }
  };

  // 删除
  const handleDelete = async (id: string, name: string) => {
    if (!confirm(`确定要删除 "${name}" 吗？`)) {
      return;
    }

    try {
      const result = await deleteItem(id);
      if (result.success) {
        await loadItems();
      } else {
        setError(result.error || '删除失败');
      }
    } catch (err) {
      setError('删除失败');
    }
  };

  return (
    <section className="space-y-6">
      <header className="flex items-center justify-between">
        <div>
          <h1 className="text-2xl font-semibold text-ink-900">{t('feature.title')}</h1>
          <p className="mt-2 text-sm text-muted">
            {t('feature.description')}
          </p>
        </div>
        {viewMode === 'list' && (
          <button
            onClick={() => setViewMode('create')}
            className="rounded-xl border border-ink-900/10 bg-surface px-4 py-2 text-sm text-ink-700 hover:bg-surface-tertiary transition-colors"
          >
            添加项目
          </button>
        )}
      </header>

      {viewMode === 'create' ? (
        // 创建/编辑表单
        <div className="space-y-4">
          <div className="grid gap-1.5">
            <label className="text-xs font-medium text-muted">名称</label>
            <input
              type="text"
              className="rounded-xl border border-ink-900/10 bg-surface-secondary px-4 py-2.5 text-sm text-ink-800 placeholder:text-muted-light focus:border-accent focus:outline-none focus:ring-1 focus:ring-accent/20 transition-colors"
              placeholder="请输入名称"
              value={itemName}
              onChange={(e) => setItemName(e.target.value)}
            />
          </div>

          <div className="grid gap-1.5">
            <label className="text-xs font-medium text-muted">描述</label>
            <input
              type="text"
              className="rounded-xl border border-ink-900/10 bg-surface-secondary px-4 py-2.5 text-sm text-ink-800 placeholder:text-muted-light focus:border-accent focus:outline-none focus:ring-1 focus:ring-accent/20 transition-colors"
              placeholder="请输入描述"
              value={itemDescription}
              onChange={(e) => setItemDescription(e.target.value)}
            />
          </div>

          {error && (
            <div className="rounded-xl border border-error/20 bg-error-light px-4 py-2.5 text-sm text-error">
              {error}
            </div>
          )}

          {success && (
            <div className="rounded-xl border border-success/20 bg-success-light px-4 py-2.5 text-sm text-success">
              保存成功
            </div>
          )}

          <div className="flex gap-3">
            <button
              className="flex-1 rounded-xl bg-accent px-4 py-2.5 text-sm font-medium text-white shadow-soft hover:bg-accent-hover transition-colors disabled:cursor-not-allowed disabled:opacity-50"
              onClick={handleSave}
              disabled={saving}
            >
              {saving ? '保存中...' : '保存'}
            </button>
            <button
              className="rounded-xl border border-ink-900/10 bg-surface px-4 py-2.5 text-sm text-ink-700 hover:bg-surface-tertiary transition-colors"
              onClick={() => setViewMode('list')}
            >
              取消
            </button>
          </div>
        </div>
      ) : (
        // 列表视图
        <>
          {loading ? (
            <div className="flex items-center justify-center py-8">
              <svg aria-hidden="true" className="w-6 h-6 animate-spin text-accent" viewBox="0 0 24 24" fill="none">
                <circle className="opacity-25" cx="12" cy="12" r="10" stroke="currentColor" strokeWidth="4" />
                <path className="opacity-75" fill="currentColor" d="M4 12a8 8 0 018-8V0C5.373 0 0 5.373 0 12h4zm2 5.291A7.962 7.962 0 014 12H0c0 3.042 1.135 5.824 3 7.938l3-2.647z" />
              </svg>
            </div>
          ) : items.length === 0 ? (
            <div className="text-center py-8">
              <p className="text-sm text-muted">暂无数据</p>
              <p className="text-xs text-muted mt-2">
                点击上方"添加项目"按钮开始
              </p>
            </div>
          ) : (
            <div className="space-y-3">
              {items.map((item) => (
                <div key={item.id} className="p-4 rounded-xl border border-ink-900/10 bg-surface">
                  <div className="flex items-start justify-between gap-3">
                    <div className="flex-1 min-w-0">
                      <h3 className="text-sm font-medium text-ink-900">{item.name}</h3>
                      <p className="mt-1 text-xs text-muted">{item.description}</p>
                    </div>
                    <button
                      className="text-xs text-muted hover:text-error p-1"
                      onClick={() => handleDelete(item.id, item.name)}
                      title="删除"
                    >
                      <svg viewBox="0 0 24 24" className="w-4 h-4" fill="none" stroke="currentColor" strokeWidth="2">
                        <path d="M18 6L6 18M6 6l12 12" />
                      </svg>
                    </button>
                  </div>
                </div>
              ))}
            </div>
          )}

          <aside className="p-4 rounded-xl bg-surface-secondary border border-ink-900/5">
            <p className="text-xs text-muted">
              <strong>提示：</strong>此功能允许您管理项目配置。
            </p>
          </aside>
        </>
      )}
    </section>
  );
}
```
