## 📦 第一步：安装（1分钟）

### 1. 下载并安装

```
1. 双击 AICowork 安装程序
2. 点击"安装"按钮
3. 等待安装完成
```

### 2. 启动应用

- 安装完成后会自动启动
- 或者在开始菜单找到 "AICowork" 点击打开

---

## 🔑 第二步：配置 API（2分钟）

### 为什么需要配置？

AICowork 支持 60+ AI 服务商，需要配置 API Key 才能使用。

### 获取 API Key

**支持的 AI 服务商包括：**
- 国内：智谱 AI、DeepSeek、阿里云通义千问、七牛云、月之暗面、华为云、MiniMax、百川智能、百度文心一言、腾讯混元、科大讯飞等
- 国外：OpenAI、Google Gemini、Cohere、Mistral AI 等
- 本地：Ollama、vLLM、LocalAI 等

**获取步骤：**
```
1. 选择支持的 AI 服务商（推荐智谱 AI 或 DeepSeek）
2. 访问对应服务商的控制台
3. 注册或登录账号
4. 在控制台创建 API Key
5. 复制生成的 API Key
```

### 在应用中配置

```
1. 打开 AICowork
2. 点击左侧栏底部的 ⚙️ 设置
3. 点击 "API 配置" 标签
4. 选择 API 类型（如：智谱 AI、DeepSeek 等）
5. 粘贴 API Key 到输入框
6. 点击"保存"按钮
```

✅ **看到提示"保存成功"就完成了！**

---

## 💬 第三步：开始对话（1分钟）

### 发送第一条消息

```
1. 在底部输入框输入文字
   例如：你好，请介绍一下你自己
2. 按 Enter 键发送
3. 等待 AI 回复
```

### 基础操作

| 操作 | 方法 |
|------|------|
| 发送消息 | 输入文字 → Enter |
| 换行 | Shift + Enter |
| 新对话 | 点击左侧 "+ 新对话" |
| 切换对话 | 点击左侧历史记录 |

### 三大核心功能

#### 1️⃣ AI 对话 💬

与 AI 大模型进行智能对话，支持：
- 多轮连续对话
- 代码编写和解释
- 文档分析和总结
- 创意写作和翻译
- 多厂商 API 支持

#### 2️⃣ 记忆系统 🧠

**本地知识库，让 AI 记住重要信息**

**使用场景：**
- 保存项目配置信息
- 记录个人偏好设置
- 存储常用代码片段
- 整理技术文档要点

**快速使用：**
```
1. 设置 → 记忆系统
2. 点击"+ 添加新记忆"
3. 填写标题和内容
4. 保存
```

#### 3️⃣ 斜杠命令 ⚡

**快捷命令，提高效率**

| 命令 | 功能 | 示例 |

## 🔥 实用技巧

### 技巧 1：使用记忆系统提高效率

```
场景：经常需要查询项目配置

步骤：
1. 将项目配置保存到记忆系统
   标题：项目根目录
   内容：C:\Users\YourName\Projects\MyApp

2. 以后随时可以：
   - 搜索"项目根目录"快速找到
   - 使用 /memory ask 让 AI 帮你查找
```

### 技巧 2：创建多个对话

```
为不同任务创建独立对话：

- "代码调试" - 专门讨论技术问题
- "文档写作" - 协助撰写文档
- "学习笔记" - 记录学习心得

点击左侧"+ 新对话"即可创建
```

### 技巧 3：利用自动记忆功能

```
设置 → 记忆系统 → 启用"自动存储记忆"

这样每次对话结束，AI 会自动分析并保存重要信息
```

## ❓ 遇到问题？

### 最常见问题

**Q: 提示"API Key 未配置"？**
```
A: 点击设置 → API 配置 → 粘贴 API Key → 保存
```

**Q: 发送消息没有反应？**
```
A: 检查网络连接，确认 API Key 正确配置
```

**Q: 如何删除历史对话？**
```
A: 点击右侧对话卡片上的删除按钮
```

**Q: 记忆搜索没结果？**
```
A: 确认已添加记忆内容，尝试使用不同的关键词
```
