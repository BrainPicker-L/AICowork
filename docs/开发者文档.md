## 项目概述

### 项目定位

DtazziCowork 是一款基于 **[@qwen-code/sdk](https://github.com/QwenLM/qwen-code)** 的现代化 AI 桌面应用，旨在为用户提供：

- 🎯 **开箱即用**: 无需复杂配置，快速上手
- 🔧 **高度可扩展**: 支持 MCP、技能、代理等扩展机制
- 🛡️ **安全可靠**: 数据本地存储，权限精细控制
- ⚡ **性能优化**: React 19 并发渲染，流畅体验

### 核心技术栈

```
前端层:
├─ React 19.2.3          # UI 框架（并发渲染）
├─ TypeScript 5.9        # 类型安全
├─ Tailwind CSS 4.1      # 原子化 CSS
├─ Zustand 5.0          # 状态管理
└─ i18next 25.7         # 国际化

桌面层:
├─ Electron 39.2.7      # 跨平台桌面框架
├─ Vite 7.3             # 构建工具
└─ electron-builder     # 打包工具

AI 层:
├─ @qwen-code/sdk 0.1.3 # AI 对话核心
├─ @memvid/sdk 2.0.120  # 向量记忆存储
└─ @modelcontextprotocol/sdk # MCP 协议支持
```

```
用户输入 → UI 组件 → Zustand Store → IPC 调用 → 主进程处理
                                                    ↓
                                              SDK Runner
                                                    ↓
                                         query() 流式响应
                                                    ↓
                                         IPC 事件推送
                                                    ↓
                                         UI 实时更新
```

### IPC 通信架构

```typescript
// 渲染进程 → 主进程
window.electron.sendMessage(prompt, sessionId);

onEvent({
  type: "stream.message",
  payload: { sessionId, message }
});

// 双向通信（请求-响应）
const config = await window.electron.getApiConfig();
```

## 核心模块详解

### 1. SDK Runner (`src/electron/libs/runner/`)

**职责**: 封装 Qwen Code SDK 的调用逻辑

**核心文件**:
- `index.ts`: 主入口，执行 SDK query
- `types.ts`: 类型定义
- `permission-handler.ts`: 权限处理
- `memory-manager.ts`: 记忆管理
- `performance-monitor.ts`: 性能监控

**关键代码**:

```typescript
import { query } from "@qwen-code/sdk";

export async function runClaude(options: RunnerOptions): Promise<RunnerHandle> {
  const { prompt, session, onEvent } = options;
  
  // 1. 获取 API 配置
  const config = await getCachedApiConfig();
  
  // 2. 构建环境变量
  const env = buildEnvForConfig(config);
  
  // 3. 加载 MCP 服务器
  const mcpServers = await getMcpServers();
  
  // 4. 创建对话流
  const conversationStream = createConversationStream(
    inputQueue,
    session.id,
    prompt
  );
  
  // 5. 执行 SDK query
  for await (const message of query(conversationStream, {
    env,
    cwd: session.cwd,
    mcpServers,
    permissionMode: "default",
    onPermissionRequest: handlePermissionRequest
  })) {
    onEvent({
      type: "stream.message",
      payload: { sessionId: session.id, message }
    });
  }
}
```

```typescript
// 获取所有 MCP 服务器配置
export async function getMcpServers(): Promise<Record<string, McpServerConfig>> {
  const configs = await loadMcpServers();
  const servers: Record<string, McpServerConfig> = {};
  
  for (const [name, config] of Object.entries(configs)) {
    // 跳过禁用的服务器
    if (config.disabled || !config.enabled) continue;
    
    // 移除 type 字段，让 SDK 自动识别
    const { type, ...cleanConfig } = config;
    servers[name] = cleanConfig;
  }
  
  return servers;
}
```

### 3. 记忆存储 (`src/electron/storage/memvid-store.ts`)

**职责**: 使用 Memvid SDK 管理向量记忆

**存储位置**: `~/.qwen/memvid/memory.mv2`

**关键代码**:

```typescript
import { use, create } from '@memvid/sdk';

class MemvidStore {
  private memory: any = null;
  
  async initialize() {
    const fileExists = await fs.access(this.filePath)
      .then(() => true)
      .catch(() => false);
    
    if (fileExists) {
      this.memory = await use('basic', this.filePath);
    } else {
      this.memory = await create(this.filePath, 'basic');
    }
  }
  
  async putDocument(input: MemoryDocumentInput) {
    const doc = await this.memory.put({
      title: input.title,
      label: input.label || 'general',
      text: input.text,
      metadata: input.metadata || {},
    });
    
    await this.memory.seal();
    return { success: true, id: doc?.id };
  }
  
  async findDocuments(query: string, options = {}) {
    const results = await this.memory.find(query, {
      mode: options.mode || 'lex',
      k: options.k || 10,
    });
    
    return { success: true, results };
  }
}
```

```typescript
export async function getSkillsList(): Promise<SkillConfig[]> {
  const skillsDir = getSkillsDir();
  const entries = await fs.readdir(skillsDir, { withFileTypes: true });
  const skills: SkillConfig[] = [];
  
  for (const entry of entries) {
    if (entry.isDirectory()) {
      const skillFile = join(skillsDir, entry.name, 'SKILL.md');
      
      try {
        await fs.access(skillFile);
        const content = await fs.readFile(skillFile, 'utf-8');
        const skill = parseSkillMarkdown(content, entry.name);
        skills.push(skill);
      } catch {
        // 跳过无效技能
      }
    }
  }
  
  return skills;
}
```

## MCP 服务器开发

### 1. 创建 MCP 服务器

使用 SDK 提供的工具快速创建：

```typescript
import { createSdkMcpServer, tool } from "@qwen-code/sdk";
import { z } from "zod";

const server = createSdkMcpServer({
  name: "my-tools",
  version: "1.0.0",
  tools: [
    tool(
      "hello",
      "打招呼",
      {
        name: z.string().describe("名字")
      },
      async (args) => {
        return {
          content: [{
            type: "text",
            text: `你好，${args.name}！`
          }]
        };
      }
    )
  ]
});

// 启动服务器
server.listen();
```

### 2. 集成到 DtazziCowork

**方式一：配置文件**

在 `~/.qwen/settings.json` 中添加：

```json
{
  "mcpServers": {
    "my-tools": {
      "command": "node",
      "args": ["./my-tools-server.js"]
    }
  }
}
```

**方式二：代码集成**

在 `src/electron/utils/` 创建服务器文件：

```typescript
// src/electron/utils/my-tools-mcp-server.ts
import { createSdkMcpServer, tool } from "@qwen-code/sdk";

export async function createMyToolsServer() {
  return createSdkMcpServer({
    name: "my-tools",
    version: "1.0.0",
    tools: [
      // 定义工具...
    ]
  });
}
```

然后在 `sdk-config-cache.ts` 中注册：

```typescript
// 获取自定义 MCP 服务器
const myToolsServer = await createMyToolsServer();

const allMcpServers = {
  'my-tools': myToolsServer,
  ...externalMcpServers
};
```

### 3. MCP 工具最佳实践

**工具命名**:
- 使用清晰的动词：`search`, `create`, `update`, `delete`
- 避免缩写：`get_user` 而非 `get_usr`

**参数设计**:
- 使用 Zod 进行类型验证
- 提供详细的描述信息
- 设置合理的默认值

**错误处理**:
```typescript
tool("risky_operation", "危险操作", schema, async (args) => {
  try {
    const result = await performOperation(args);
    return {
      content: [{ type: "text", text: result }]
    };
  } catch (error) {
    return {
      content: [{
        type: "text",
        text: `操作失败: ${error.message}`
      }],
      isError: true
    };
  }
});
```

1. 通过 UI：设置 → 记忆管理 → 清空记忆
2. 手动删除：`rm ~/.qwen/memvid/memory.mv2`

### 4. 开发调试

**Q: 如何查看渲染进程日志？**

A: 按 `F12` 打开 DevTools，查看 Console

**Q: 如何重新加载应用？**

A: 按 `Cmd+R` (macOS) 或 `Ctrl+R` (Windows/Linux)

**Q: 如何清除应用数据？**

A: 删除用户数据目录：

```bash
# macOS
rm -rf ~/Library/Application\ Support/dtazzicowork

# Windows
rmdir /s %APPDATA%\dtazzicowork

# Linux
rm -rf ~/.config/dtazzicowork
```
